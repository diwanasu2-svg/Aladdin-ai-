# Porcupine Wake-Word Model Files

Place the following files in this directory.  
Obtain them from the [Picovoice Console](https://console.picovoice.ai/):

| File | Description |
|------|-------------|
| `porcupine_params.pv` | Porcupine parameters file (downloaded from Picovoice) |
| `Aladdin.ppn` | Custom "Aladdin" wake-word model (trained in Picovoice Console) |
| `porcupine_license.key` | Access key (also set `PORCUPINE_ACCESS_KEY` in `secrets.properties`) |

These files are **excluded from git** (see `.gitignore`).  
Do NOT commit them to version control.
