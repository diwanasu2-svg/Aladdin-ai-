# POST-INSTALL NOTES
# ════════════════════════════════════════════════════════════════════════════
#
# 1. Generate a JWT secret key:
#    python -c "import secrets; print(secrets.token_hex(32))"
#    Set as: JWT_SECRET_KEY=<output>
#
# 2. Generate an encryption key:
#    python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
#    Set as: ENCRYPTION_KEY=<output>
#
# 3. Set required environment variables:
#    JWT_SECRET_KEY          — 32+ byte random hex secret (REQUIRED)
#    ENCRYPTION_KEY          — Fernet-format base64url key (REQUIRED)
#    AUDIT_SIGNING_KEY       — HMAC signing key for audit log tamper detection
#    AUDIT_LOG_PATH          — Path for audit log file (default: logs/audit.log)
#    AWS_ACCESS_KEY_ID       — For AWS Secrets Manager (optional)
#    AWS_SECRET_ACCESS_KEY   — For AWS Secrets Manager (optional)
#    VAULT_ADDR              — HashiCorp Vault address (optional)
#    VAULT_TOKEN             — HashiCorp Vault token (optional)
#    SENTRY_DSN              — Sentry crash reporting URL (optional)
#
# 4. Install:
#    pip install -r requirements_phase12_phase13.txt
#
